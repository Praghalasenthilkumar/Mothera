Mothera is a modern, elegant pregnancy and motherhood companion web app built on the Caffeine platform (Internet Computer). It supports expectant and new mothers with personalized tracking, reminders, wellness resources, emergency assistance, and an AI-powered chat assistant named Tina.

The app offers two distinct journeys — Pregnancy and Having Baby — each with its own dashboard, modules, and layout, but both unified by a strict purple-and-white color theme.

Pregnancy journey includes an interactive 10-month circular timeline with animated 2D baby growth illustrations (M1–M10), medicine reminders with before/after food details and edit options, checkup reminders, Emergency SOS, Doctor Connect (OB-GYN, nutritionist, physiotherapist), and a wellness library with real photos of pregnant women doing exercise, yoga, and meditation.

Having Baby journey features Mother's Wellness (with after-birth and newborn care videos), medicine and vaccine reminders, and an enhanced Doctor Connect with a child specialist. Baby milestone, emergency assist, and nearby help modules were removed from this flow.

Tina, the AI assistant, greets users with a single welcome message and offers text and voice chat restricted to pregnancy-friendly tips. It opens in a dedicated page with the typing area above the menu bar.

Other modules include Community (separate Mothers' and Baby Topics channels with messaging and stories), Shopping (only mother and baby products with real images, linking to Amazon), and Classes (expanded catalog with real instructor icons).
